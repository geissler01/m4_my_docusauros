# Prácticas de Bases de Datos

Este espacio documenta las actividades y ejercicios realizados durante el proceso de estudio de bases de datos relacionales, utilizando principalmente MySQL y Postgres como sistema de gestión.

El objetivo es consolidar fundamentos técnicos, desarrollar pensamiento lógico aplicado a datos y construir una base sólida para backend y análisis de información.

## Propósito

- Comprender el funcionamiento de bases de datos relacionales.
- Dominar consultas SQL desde nivel básico hasta avanzado.
- Aplicar buenas prácticas en modelado y análisis de datos.
- Desarrollar criterio técnico al trabajar con información estructurada.

## Enfoque de trabajo

Las actividades estarán organizadas por niveles de dificultad y cubrirán temas como:

- Consultas básicas (`SELECT`, `WHERE`)
- Operadores lógicos y filtros compuestos
- Funciones de agregación (`COUNT`, `AVG`, `SUM`)
- Agrupaciones con `GROUP BY`
- Subconsultas
- Clasificaciones con `CASE`
- Análisis comparativos y rankings
- Optimización y buenas prácticas

Cada actividad busca no solo resolver consultas, sino entender el razonamiento detrás de cada solución.

## Objetivo final

Construir una comprensión sólida de bases de datos que permita:

- Diseñar estructuras eficientes.
- Consultar información de manera óptima.
- Interpretar resultados correctamente.
- Aplicar estos conocimientos en proyectos reales.
